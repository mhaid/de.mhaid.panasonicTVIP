Diese App bietet eine IP-Fernbedienung für Ihren Panasonic Viera Fernseher.
Es werden nur TVs von 2011 bis 2018 unterstützt, da es die Panasonic TV Remote 2 App von Panasonic Corp. emuliert!
Ihr Homey muss mit dem Netzwerk verbunden sein, mit dem Ihr Fernseher verbunden ist.

Die App verfügt über eine Automatische Gerätesuche; Falls diese Ihr Gerät nicht erkennt, kann dieses manuell hinzugefügt werden.

Wenn zusätzlich zur IP auch die MAC-Addresse des Fernsehers angegeben wurde, sucht Homey im Hintergrund nach einer Änderung der IP des Fernsehers.


Um die beste Nutzererfahrung und Stabilität zu erreichen, wird eine statische IP-Addresse Empfohlen. Dies kann in den Einstellungen Ihres Fernsehers eingesellt werden.


Kommando-Anfragen basierend auf der tollen Arbeit von m4recek: https://github.com/m4recek/panasonic-viera-remote-control

Wake On Lan basierend auf der tollen Arbeit von agnat: https://github.com/agnat/node_wake_on_lan
